About This Project

What is gh-intro?

A minimal, elegant GitHub introduction page inspired by Claude.ai's design language. It serves as your digital front door — a warm, approachable way to introduce yourself and your work to the developer community.

Tech Stack

Built with zero dependencies — pure HTML, CSS, and JavaScript. No build tools, no frameworks, no complexity. Just clean, semantic markup with Claude.ai-inspired warmth and scroll-driven animations.

Features

Responsive design, smooth scroll animations, animated counters, hover micro-interactions, dark/light aware, and GitHub Pages ready. Everything you need to make a great first impression.

How to Use This Repo

Get your own GitHub introduction page up and running in under 5 minutes.

1

Clone the repository

bash 
git clone https://github.com/yourusername/gh-intro.git
cd gh-intro
2

Customize your content

Edit index.html to update your name, bio, projects, and links. Replace the avatar URL with your own GitHub avatar.

url 
https://avatars.githubusercontent.com/u/YOUR_ID?v=4
3

Preview locally

Open index.html directly in your browser, or use a simple local server:

bash 
python3 -m http.server 8000
4

Deploy to GitHub Pages

Push to your GitHub repository, then go to Settings → Pages and select the main branch as source. Your site will be live at:

url 
https://yourusername.github.io/gh-intro/
